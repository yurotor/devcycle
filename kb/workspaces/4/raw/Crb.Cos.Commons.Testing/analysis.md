# Crb.Cos.Commons.Testing

## Purpose
A .NET testing utility library providing tools for unit and integration testing in COS services

## Business Features
- Behavior-Driven Development (BDD) testing framework
- Mocking of repositories and database access
- Validation utilities for test assertions
- NHibernate mapping validation

## Data Entities
- **Generic repositories** — Support for mocking IRepository<T> and ICachingRepository<T> interfaces from Afterman.nRepo
