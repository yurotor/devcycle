# Crb.Cos.Commons

## Purpose
Shared utility library providing common functionality for .NET applications in the Crb.Cos ecosystem

## Business Features
- Banking utilities (date formatting, check digit validation, routing number validation)
- Caching abstractions and implementations
- Configuration management
- Database connection management (SQL Server and PostgreSQL)
- Date handling with timezone support
- Holiday calculations
- File system operations
- String and object extensions
- Secret management (AWS Secrets Manager integration)
- IoC container support (Autofac)
- Object mapping utilities
- GUID generation

## Dependencies
- **AutoMapper** (shared-lib)
- **Autofac** (shared-lib)
- **LazyCache** (shared-lib)
- **Microsoft.Extensions.Caching.Memory** (shared-lib)
- **Newtonsoft.Json** (shared-lib)
- **Npgsql** (shared-lib)
- **System.Data.SqlClient** (shared-lib)
