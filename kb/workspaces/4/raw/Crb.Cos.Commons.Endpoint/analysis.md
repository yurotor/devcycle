# Crb.Cos.Commons.Endpoint

## Purpose
Provides a standardized way to configure and bootstrap .NET host applications with common concerns like dependency injection, messaging, logging, configuration management, and health checks.

## Business Features
- Standardized host creation and bootstrapping for backend services
- Environment-variable based configuration loading
- RabbitMQ health check implementation
- NewRelic integration for application monitoring
- Support for secrets management
